# 📝 Task Manager Application

A full-stack Task Manager web application that helps users efficiently manage their daily tasks with authentication and real-time updates.

---

## 🚀 Features

- 🔐 User Authentication (Login / Signup)
- 📋 Create, Read, Update, Delete Tasks
- ✅ Task Status (Pending / Completed)
- 📊 Dashboard with task statistics
- ⚡ Fast and responsive UI
- 🔒 Secure backend APIs

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Axios
- React Router

### Backend
- Node.js
- Express.js
- MongoDB
